# Front-end Dev
My front end stack that changes day to day :)

## Installation

### Install the necessary package managers

npm install -g bower

npm install -g gulp-hologram

gem install bundler

### Clone the repo
git clone git@github.com:heyMP/frontend-dev.git

cd frontend-dev

### Install the plugins
npm install

bower install

bundle install

### Run Gulp Serve
gulp serve
